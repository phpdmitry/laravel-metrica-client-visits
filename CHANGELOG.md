# Changelog

## v0.2.0 — 2026-07-23

- Сохраняются и выдаются все найденные визиты ClientID через `VisitCandidate`.
- Повторный завершённый или failed запрос с тем же `externalId` создаёт новую выгрузку.
- Добавлена объединённая история кандидатов по `externalId` без дублей `visit_id`.
- Добавлена необязательная стратегия выбора `first`; прежняя стратегия `last` остаётся значением по умолчанию.
